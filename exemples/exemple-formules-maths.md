## Question 1 : Quelle est la dérivée de $f(x) = 3x^2 + 5x - 7$ ?
- [ ] $f'(x) = 3x + 5$
- [x] $f'(x) = 6x + 5$
- [ ] $f'(x) = 6x^2 + 5$
- [ ] $f'(x) = 6x - 7$
> Explication: On applique la règle $\frac{d}{dx}x^n = nx^{n-1}$ à chaque terme : $3 \cdot 2x + 5 = 6x + 5$.
> Points: 20

## Question 2 : Quelle est la valeur de l'intégrale suivante ? $$\int_0^1 2x \, dx$$
- [x] $1$
- [ ] $2$
- [ ] $0$
- [ ] $\frac{1}{2}$
> Explication: $$\int_0^1 2x \, dx = \left[ x^2 \right]_0^1 = 1^2 - 0^2 = 1$$
> Points: 30

## Question 3 : Simplifiez l'expression suivante : $$\sum_{k=1}^{n} k = \frac{n(n+1)}{2}$$ Pour $n = 100$, combien vaut cette somme ?
= 5050
> Explication: $$\sum_{k=1}^{100} k = \frac{100 \times 101}{2} = \frac{10100}{2} = 5050$$ Cette formule est attribuée à Gauss, qui l'aurait trouvée enfant.
> Points: 50
